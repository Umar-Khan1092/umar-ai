# Traffic Speed Tracking System

## Overview
AI-based system for detecting and calculating vehicle speed in real time.

## Problem
Manual speed monitoring was inefficient and inaccurate.

## Solution
Implemented computer vision models using YOLO and OpenCV for live speed calculation.

## Technologies Used
- Python  
- YOLO  
- OpenCV  

## Key Features
- Real-time speed detection  
- Vehicle tracking  
- Automated reports  

## Results & Impact
- Improved traffic monitoring  
- Accurate speed analytics
