# n8n Automation WhatsApp Agent

## Overview
An AI system combining n8n automation and whatsapp agent for automating resturant ordering.

## Problem
Manual ordering was time-consuming and error-prone.

## Solution
Automated the ordering process using n8n automation and whatsapp agent.

## Technologies Used
- n8n 
- Whatsapp Business API  
- Python  

## Key Features
- Automated ordering process  
- Real-time order updates  
- Real-time support  

## Results & Impact
- Reduced manual work  
- Improved system reliability
