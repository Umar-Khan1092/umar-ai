# AI Document Chatbot System

## Overview
An AI chatbot allowing users to upload documents and interact with them through natural language queries.

## Problem
Users struggled to extract insights from long documents quickly.

## Solution
Developed an NLP-powered chatbot using large language models to answer questions based on uploaded content.

## Technologies Used
- Python  
- NLP Models  
- Vector Databases  
- Web Interface  

## Key Features
- Document upload support  
- Context-aware responses  
- Fast information retrieval  

## Results & Impact
- Improved productivity  
- Faster data access  
- Enhanced user experience
