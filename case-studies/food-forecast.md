# Food Demand Forecasting & Procurement System

## Overview
AI-driven forecasting system to predict food demand and optimize procurement.

## Problem
Manual forecasting caused overstock and food waste.

## Solution
Implemented time-series machine learning models to predict demand accurately.

## Technologies Used
- Python  
- Forecasting Models  
- Data Analytics Tools  

## Key Features
- Demand prediction dashboards  
- Automated procurement planning  
- Trend analysis  

## Results & Impact
- Reduced waste  
- Improved inventory control  
- Cost savings
