# Mental Health Detection System

## Overview
AI-powered system designed to assess mental health indicators using facial expressions, behavioral patterns, micro facial expressions, and data analytics.

## Problem
Manual mental health screening is time-consuming and inconsistent.

## Solution
Built deep learning and computer vision models to analyze emotional and behavioral signals for automated assessment.

## Technologies Used
- Python  
- Deep Learning  
- Computer Vision 
- React JS  

## Key Features
- Emotion detection  
- Behavioral pattern analysis  
- Real-time processing  

## Results & Impact
- Faster screening process  
- Improved consistency  
- Data-driven insights
