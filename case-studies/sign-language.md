# Greek Sign Language Recognition System

## Overview
AI system for recognizing Greek sign language gestures across different signers.

## Problem
Communication barriers for sign language users in digital systems.

## Solution
Trained deep learning models on signer dependent and independent datasets with gloss-level recognition.

## Technologies Used
- Python  
- Deep Learning  
- Computer Vision  

## Key Features
- High recognition accuracy  
- Multi-signer support  
- Gloss-level predictions  

## Results & Impact
- Improved accessibility  
- Reliable gesture recognition
