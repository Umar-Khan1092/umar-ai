# QR Code Ordering Automation System

## Overview
Restaurant automation system allowing customers to place orders using QR menus.

## Problem
Manual order taking slowed service and caused errors.

## Solution
Built a QR-based ordering platform with real-time backend processing.

## Technologies Used
- Python  
- Web Development  
- Database Systems  

## Key Features
- QR menu scanning  
- Automated order flow  
- Live kitchen updates  

## Results & Impact
- Faster service  
- Reduced errors  
- Improved customer experience
